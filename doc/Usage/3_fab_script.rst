Fab scripts
===========


.. topic:: Overview

    Several Fab scripts have been provided. You can simply use it to
    run the unittest, coverage and flake8

Unit test
---------
Run the unit tests.

.. code-block:: console

    $ fab test

Coverage
--------
Check tests coverage.

.. code-block:: console

    $ fab coverage

PEP8 check
----------
PEP8 check for the source code.

.. code-block:: console

    $ fab flake8
